Pizza Sales Analysis using SQL
 Project Overview

This project demonstrates my ability to analyze real-world business data using SQL by working on a comprehensive Pizza Sales dataset. The goal was to extract actionable business insights related to sales performance, customer preferences, revenue trends, and product demand

Business Objectives

-The analysis focuses on answering key business questions such as:

-How many orders were placed?

-How much revenue was generated?

-Which pizzas and categories perform best?

-What are customer ordering patterns by size, category, and time?

-How does revenue grow over time?

Tools & Technologies Used

-SQL (MySQL)

-Relational database concepts

-Aggregate functions (SUM, COUNT, AVG)

-Joins (INNER JOIN)

-Window functions (RANK() OVER)

-Subqueries

-Time-based analysis (HOUR, ORDER BY date)

-PowerPoint (Canva-style presentation for storytelling)
